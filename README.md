# AwesomeWeather
Awesome weather app

##Design patters used: MVVM

##How to run the app?

cd to AwesomeWeather
pod install

Hit run on Xcode after selecting any iPhone simulator

##How to run tests

Run the unit test case suite in the AwesomeWeatherTests target.
Hit CMD + U in Xcode to test test cases
